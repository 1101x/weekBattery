
1. 개요
	1. 일주일 예산 관리 가계부 웹앱
2. 주요 키아트
	1. 3d 형태의 단순한 배터리 아이콘이 있다
	2. 전체 예산 금액 100%에서 사용한 금액만큼 배터리가 줄어든다
3. 페이지구성
	1. 초기 화면
		1. 가운데 단순한 쓰리디 배터리
		2. 배터리 하단에 내역 추가 버튼
	2. 예산 설정 화면
	3. 사용 내역 1주일 기준 화면
4. 디자인 톤앤매너
	1. 매우 심플함
	2. 네온 그린 배터리
5. 세부 요소
	1. 배터리 : 터치하여 회전시킬수잇음
	2. 문자를 복사 붙여넣기하여 소비 항목 금액을 메모 할 수 있다
